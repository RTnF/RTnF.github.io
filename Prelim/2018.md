## 国内予選2018
### A (100) - 所得格差 {#A}
#### 問題概要
$n$ 項の数列 $a_n$ のうち、平均値以下である項の数はいくらか。

#### 制約
- 入力はすべて整数
- 複数ケース ( $\sum n \le 50000$ )
- $2 \le n \le 10000$
- $1 \le a_i \le 100000$

#### メモ
$S = \sum_{i} a_i$ とする。整数で処理したいので、条件式 $a_i \le S/n$ を $n a_i \le S$ に変形する。
