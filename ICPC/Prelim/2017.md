## 国内予選2017
### A (100) - 太郎君の買物
#### 問題概要
$n$ 個の商品からちょうど2つ選ぶ。各商品の価格は $a_i$ 円である。  
価格の合計が $m$ 円以下である組み合わせのうち、価格の合計の最大値を求めよ。

#### 制約
- 入力はすべて整数
- 複数ケース ( $\sum n \le 50000$ )
- $2 \le n \le 1000$
- $2 \le m \le 2,000,000$
- $1 \le a_i \le 1,000,000$

#### メモ
素直に二重ループで解ける。分かりにくい点は同じ商品2個がNGであること。
